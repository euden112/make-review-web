# Frontend

## 구현 사항
- 게임 목록 페이지 (임시 데이터로 게임 카드 표현)
- 게임 상세 페이지 (리뷰 요약 블록)
- 페이지 간 라우팅

## 로컬 실행 방법
1. 이 레포지토리 클론
   git clone https://github.com/euden112/make-review-web.git

2. frontend 폴더로 이동
   cd make-review-web/frontend

3. 패키지 설치
   npm install

4. 개발 서버 실행
   npm run dev

5. 브라우저에서 접속
   http://localhost:5173

## Vercel 배포 방법 (팀장용)
1. vercel.com 접속 → GitHub 계정으로 로그인
2. Add New Project 클릭
3. make-review-web 레포지토리 선택 후 Import 클릭
4. Root Directory 설정에서 frontend 선택
5. Deploy 클릭
6. 배포 완료 후 생성된 URL을 아래에 추가
